## GarbageBoy
Work in progress.

A garbage (haha) Gameboy emulator written in C++.

### Building

#### Install Dependencies
##### Debian Linux
```bash
sudo apt install libboost-all-dev libsdl2-dev
```

##### OSX
```bash
brew install boost sdl2
```

#### Compile
You can just run `make`.
